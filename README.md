# practica-tdd

Proyecto para practicar TDD.

La aplicación clasifica triángulos según la longitud de sus lados y devuelve uno de estos resultados:

- `Equilatero`
- `Isosceles`
- `Escaleno`
- `NoTriangulo`

El proyecto incluye pruebas unitarias con JUnit para validar esos casos.

## Ejecutar pruebas

```bash
mvn test
```
